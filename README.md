# Excel2Code
Excel To  Code

### Environment
python-3.10.4

pip install xlrd==1.2.0

pip install pyyaml


### __Path.yaml    
basic configuration
Path:
  CodePath: ../Data/Code
  XlsxPath: ../Config
  BinPath: ../Data/Bin
  CodeType: csharp   

### Run
Use exceltool.sh




Currently only supports CSharp
